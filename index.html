memory allocator - C


a dynamic memory allocation program that imitates the allocation functions in C. has unique implementations ufree and umalloc which mirror free and malloc C system calls. stored on heap and tested by different checks to the heap.

system emulator - C


a virtualization of system processes including an implementation of the RISC pipeline, including forwarding of values and hazard controls. the system itself is able to execute Y86 instructions, a simplified form of standard x86 assembly instructions.
