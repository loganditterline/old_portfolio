My name is Logan Ditterline and I am currently a Computer Science major at the University of Texas at Austin. I'm in my sophomore year and this is a quick collection of my major projects.

Sudoku Solver - Python


A project that generates and solves sudoku puzzles. The user can choose how many tiles are removed from the puzzle and how quickly it runs. The program has a ui to show the backtracking algorithm at work.

Tthe program itself was made with the pygame library so it cannot be demo'd here.

Memory Allocator - C


A dynamic memory allocation program that imitates the allocation functions in C. Has unique implementations ufree and umalloc which mirror free and malloc C system calls. Stored on heap and tested by different checks to the heap.

System Emulator - C


A virtualization of system processes including an implementation of the RISC pipeline, including forwarding of values and hazard controls. The system itself is able to execute Y86 instructions, a simplified form of standard x86 assembly instructions.
